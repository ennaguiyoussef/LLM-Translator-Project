# LLM-Translator-Project

## Description
This project implements an LLM-powered RESTful web service for translating English into Moroccan Arabic (Darija). The service exposes a REST API and integrates a large language model (e.g., Google Gemini or similar) to generate translations. The system also includes a PHP client and a Chrome extension (Manifest V3) for user-facing access, and it secures the API using Basic Authentication.
